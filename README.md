# dks_console
A simple CLI resource for students of the Chudokai Aikido Federation International for Dan Koto Shitsumon.

## How to use
1. clone this repo to a local directory
2. enter the directory and run the command '''python3 main.py'''
3. follow the instructions in your console

## Advanced Use
You can use the convertCSV.py file to convert a .csv file to a .json file and add your own questions.

1. upload your .csv file to the main directory and rename to "dks.csv"
2. in your terminal, run the command '''python3 convertCSV.py'''
3. the new file will be generated for you. you can then run the main.py file
